## 🎨 Interactive Computer Graphics
This project demonstrates **24-bit Sprite Transformation** using WebGL / Canvas API.

### 🎮 How to interact:
- **Move (Translate):** Use `Arrow Keys` to move the sprite around.
- **Rotate:** Press `R` to rotate the sprite clockwise.
- **Scale:** Press `S` to scale up and `D` to scale down.
